# Sky Studio 解密工具 - 微信小程序版（原生移动端风格）

## 📱 项目简介

这是 Sky Studio 解密解析工具的**原生微信小程序版本**，专为移动端优化设计。

### ✨ 特性

- 🎨 **原生移动端设计**：采用微信小程序原生组件和交互方式
- 📱 **手势友好**：侧边栏滑动打开/关闭，适配触控操作
- 🎵 **音频播放**：支持播放转换后的乐谱（使用原项目的 sounds 资源）
- 🖼️ **导出图片**：可以将乐谱导出为图片保存到相册
- 💾 **文件下载**：支持将代码文件分享保存
- 🌙 **深色模式**：支持深色/浅色主题切换
- 💾 **本地存储**：使用 wx.setStorage 持久化数据

## 🚀 使用微信开发者工具

### 步骤1：安装微信开发者工具

1. 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载并安装对应操作系统的版本
3. 打开微信开发者工具

### 步骤2：导入项目

1. 点击 "+" 或"导入项目"按钮
2. 填写项目信息：
   - **项目名称**：`Sky Studio解密工具`
   - **目录**：选择 `C:\Users\user\skystudio-miniapp`
   - **AppID**：
     - 测试：选择"测试号"（推荐，无需注册）
     - 正式：填写你的小程序 AppID
3. 点击"导入"

### 步骤3：预览和调试

#### 在模拟器中预览
- 导入后会自动编译
- 左侧是模拟器，可以看到界面
- 可以切换不同设备尺寸

#### 真机预览
1. 点击工具栏"预览"按钮
2. 用微信扫描二维码
3. 在真实手机上体验

#### 本地调试
- 使用调试器查看日志
- 可以在代码中打断点调试

## 📖 使用说明

### 1️⃣ 上传文件

**操作**：
- 点击左上角 `☰` 打开侧边栏
- 点击 `📁 上传文件` 按钮
- 选择 `.txt` 格式的 Sky Studio 工程文件

**提示**：
- 支持一次选择多个文件
- 文件会自动保存到本地列表

### 2️⃣ 查看和管理项目

**文件列表**：
- 在侧边栏可以看到所有上传的文件
- 点击文件可以激活查看详情
- 选中的文件会显示绿色边框

**批量操作**：
- `☑️/☐`：全选/取消全选
- `🗑️`：清空列表
- 每个文件前的复选框：选择/取消选择

**删除项目**：
- 点击文件右边的 `🗑️` 按钮
- 确认后删除

### 3️⃣ 编辑乐谱信息

激活项目后，在侧边栏下方可以看到"🎵 乐谱信息"：

- **名称**：乐谱名称
- **作者**：作者名称
- **BPM**：节拍速度（1-999）
- **改编者**：改编者名称

**提示**：编辑后会自动重新生成代码

### 4️⃣ 播放预览

在主界面可以看到"▶️ 播放"按钮：

**控制功能**：
- `▶️/⏸️`：播放/暂停
- `⏹️`：停止
- `🏃 倍速`：调整播放速度（0.25x - 3x）
- **进度条**：显示播放进度
- **音符计数**：显示当前播放位置

**提示**：需要 `sounds` 文件夹中的音频文件才能播放

### 5️⃣ 获取转换结果

在项目激活且转换完成后，侧边栏底部会出现操作按钮：

- `📋 复制代码`：复制生成的代码到剪贴板
- `💾 下载代码`：将代码保存为文件并分享
- `🖼️ 导出图片`：将乐谱导出为图片

#### 下载功能说明

点击"💾 下载代码"后：
1. 代码会保存为 `.txt` 文件
2. 会调用微信的"分享文件"功能
3. 你可以选择：
   - 发送给朋友
   - 保存到微信收藏
   - 用其他应用打开
4. 成功后会提示文件保存位置

### 6️⃣ 导出图片

点击"🖼️ 导出图片"后：
1. 会在 canvas 上绘制简谱
2. 绘制完成后可以：
   - 点击"💾 保存图片"保存到相册
   - 点击"❌ 关闭"关闭预览

### 7️⃣ 主题切换

- 点击右上角图标：
  - `🌙`：当前是浅色模式，点击切换到深色
  - `☀️`：当前是深色模式，点击切换到浅色

## ⚠️ 注意事项

### 1. 音频播放

- 需要 `sounds` 文件夹中的音频文件
- 已复制到小程序目录
- 微信小程序音频播放有限制，需用户交互触发

### 2. 文件下载

- 微信小程序不支持直接下载到本地文件系统
- 使用 `wx.shareFileMessage` 分享文件
- 用户可以保存到微信收藏或用其他应用打开

### 3. 本地存储

- 使用 `wx.setStorageSync` 代替 `localStorage`
- 数据持久化到微信本地缓存
- 清除小程序缓存会丢失数据

### 4.  canvas 导出图片

- 需要用户授权保存图片到相册
- 如果授权失败，可以提示用户手动保存

### 5.  AppID 限制

- 测试号只能在开发者工具中预览
- 要发布需要申请正式的 AppID
- 申请地址：https://mp.weixin.qq.com/

## 🐛 常见问题

### Q1: 为什么上传文件没反应？

**A**: 检查文件格式是否为 `.txt`，以及文件内容是否正确。

### Q2: 播放没有声音？

**A**: 检查 `sounds` 文件夹是否存在，以及音频文件是否完整。

### Q3: 下载的文件在哪里？

**A**: 微信会调用分享功能，你可以选择保存到微信收藏，或用其他应用打开。

### Q4: 导出的图片在哪里？

**A**: 如果授权了相册权限，会在相册中。否则需要长按图片保存。

### Q5: 数据丢失了？

**A**: 检查是否清除了小程序缓存。数据保存在本地缓存中。
